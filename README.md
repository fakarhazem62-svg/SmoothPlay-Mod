# 🎮 SmoothPlay Mod — v2.0

مود Fabric واحد يجمع **كل** تحسينات الأداء والمظهر.

## ✨ الميزات

| الميزة | التفاصيل | تأثير الـ FPS |
|---|---|---|
| 🎨 **توضيح الألوان 200%** | Shader مدمج | — |
| ✋ **إلغاء اهتزاز المشي** | `bobView` ملغي كلياً | — |
| 💥 **إلغاء اهتزاز الضرر** | لا تميل الشاشة عند الضرب | — |
| 🚫 **إلغاء الجسيمات الزائدة** | Cap عند 256 جسيم (الافتراضي 16,000!) | **+30% FPS** |
| ☁️ **إيقاف الغيوم** | أثقل عنصر في rendering السماء | **+15% FPS** |
| 🌧️ **إيقاف المطر/الثلج** | أثقل rendering في اللعبة | **+25% FPS** |
| 👾 **إلغاء رسم الكيانات البعيدة** | لا يُرسم شيء خارج 64 بلوك | **+20% FPS** |

| 📐 **Simulation Distance = 6** | أقل كيانات تُحسب | **+10% FPS** |
| 🎭 **Fast Graphics Mode** | أقل تفاصيل في الأوراق والماء | **+15% FPS** |

> **المجموع التقريبي: +80~120% FPS** مقارنة بالافتراضي

## ✅ الإصدارات المدعومة
- Minecraft Java **1.20.1 / 1.21 / 1.21.1**
- Fabric Loader **0.15+**

## 📦 تحميل الـ JAR

اضغط على **Actions** أعلى الصفحة ← آخر build ← **Artifacts** ← حمّل `smoothplay-*.jar`

## 🔧 التثبيت

1. حمّل [Fabric Loader](https://fabricmc.net/use/installer/)
2. ضع `smoothplay.jar` في مجلد `.minecraft/mods/`
3. شغّل اللعبة — لا تحتاج أي إعداد إضافي

## 📱 على الجوال (PojavLauncher)

1. ثبّت Fabric في PojavLauncher
2. ضع الـ JAR في `mods/`
3. شغّل — كل شيء يعمل تلقائياً

## ⚙️ كيف يعمل

المود يستخدم **Fabric Mixins** لاختراق كود اللعبة مباشرة وإلغاء العمليات الثقيلة:
- **Particle cap**: يمنع اللعبة من رسم أكثر من 256 جسيم
- **Weather skip**: يلغي render المطر والثلج تماماً
- **Entity distance cull**: يتجاهل رسم أي entity أبعد من 64 بلوك
- **Settings enforcement**: كل ثانية، يُعيد ضبط الإعدادات للوضع الأمثل
