# 🎮 SmoothPlay Mod

مود Fabric واحد يجمع كل شيء:

| الميزة | التفاصيل |
|---|---|
| 🎨 **توضيح الألوان** | تشبيع 200% عبر shader مدمج |
| ✋ **إلغاء اهتزاز المشي** | الشاشة ثابتة أثناء الحركة |
| 💥 **إلغاء اهتزاز الضرر** | لا تميل الشاشة عند الضرب |
| 🎯 **إيم ثابت** | يُفرض في كل tick — لا يهتز أبداً |
| 🚀 **رفع FPS** | تعطيل entity shadows وتحسين الرسم |

## ✅ الإصدارات المدعومة
- Minecraft Java 1.20.1
- Minecraft Java 1.21
- Minecraft Java 1.21.1
- Fabric Loader 0.15+

## 📦 تحميل الـ JAR

> اضغط على **Actions** أعلى الصفحة → آخر build → **Artifacts** → حمّل `smoothplay-*.jar`

## 🔧 التثبيت

1. حمّل [Fabric Loader](https://fabricmc.net/use/installer/)
2. ضع `smoothplay.jar` في مجلد `.minecraft/mods/`
3. شغّل اللعبة

## 📱 على الجوال (PojavLauncher)

1. ثبّت Fabric عبر PojavLauncher
2. ضع الـ JAR في مجلد `mods/`
3. شغّل الـ profile
